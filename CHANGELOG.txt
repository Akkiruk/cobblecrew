## 2.1.0
- Phase 1: Base class infrastructure for V2 job system
- PokemonProfile: eligibility caching computed once per Pokémon on pasture entry
- WorkerDispatcher: profile-based job assignment with stickiness + isAvailable filtering
- JobConfig + JobConfigManager: per-job JSON config under config/cobbleworkers/
- New base classes: BaseProducer, BaseSupport, BaseProcessor, BasePlacer, BaseDefender
- DSL builders: GatheringJob, ProductionJob, ProcessingJob, PlacementJob, DefenseJob, SupportJob
- Registry categories: LegacyJobs + 9 category stubs for future DSL-based jobs

## 2.0.0
- V2 Architecture: BlockCategory-based scanner, WorkerPriority tiers, WorkerRegistry
- All 22 jobs declare V2 metadata (name, targetCategory, isEligible)
- DeferredBlockScanner: fixed config caching bug, added chunk-loaded guard, dual category+legacy scanning
- BaseHarvester: container overflow timeout (30s), harvestTool parameter
- CobbleworkersCacheManager: BlockCategory API, removeTargetGlobal for overlapping pastures
- CobbleworkersNavigationUtils: mob targeting for future combat jobs
- CobbleworkersInventoryUtils: barrel extraction methods (findInputContainer, extractFromContainer)
- WorkerDispatcher: now uses WorkerRegistry instead of hardcoded list

## 1.9.0
- Added visual feedback: Pokémon swing hands, look at targets, spawn particles, and cry when working
- Added one-job-at-a-time system: Pokémon focus on completing one task before switching jobs
- Added EXCITED behavior flag while working and return-to-pasture when idle
- Refactored all harvesters to use BaseHarvester class (eliminated 161 lines of duplicate code)
- All workers now have 1.5-second work delays with animations at targets

## 1.8.0
- Fixed an issue with combee evolution line not generating honey if game was in a different language.
- By default, irrigators now irrigate a 3x3 area (9 blocks). This setting is configurable.
- By default, fire extinguishers now extinguish a 3x3 area (9 blocks). This setting is configurable.
- By default, Ditto will now handle all jobs instead of Pikachu. (Old configs will keep Pikachu)
- Vivichoke is now replanted.
- Added support for hearty grains and galarica nut bush.
