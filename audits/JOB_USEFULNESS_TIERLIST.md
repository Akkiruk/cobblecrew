# Job Usefulness Tier List

How much does each job actually help a player on a Cobblemon SMP?

### Ranking Criteria
- **Resource demand** — How often does the player need this item/effect?
- **Manual tedium** — How annoying is it to get by hand?
- **Replaceability** — Can a hopper/redstone farm already do this?
- **Cobblemon relevance** — Extra weight for Pokéball/EV/competitive resources
- **Uniqueness** — Does this do something no other automation can?

---

## S — Game-Changing (Combo Only)

These are the prestige payoffs for dedicating 2+ move slots. Each one does something
no single-move job can replicate, and all of them save massive amounts of time or
provide effects that are otherwise impossible.

| Job | Combo | Why S-Tier |
|---|---|---|
| CM13 Vein Miner | earthquake + dig | Breaks entire ore/stone veins in one go. The #1 time-saver in the mod |
| CM14 Tree Feller | cut + headbutt | Fells an entire tree at once. Wood farming goes from minutes to seconds |
| CM11 Fortune Miner | powergem + dig | Doubled ore drops without needing the enchant. Diamonds, lapis, coal — all doubled |
| CM12 Silk Touch Extractor | psychic + gathering move | Harvests blocks in block-item form. Grass blocks, spawners(?), glass, ice — without the enchant |
| C11 Blast Furnace | overheat + ironhead | 2x smelting speed + processes ancient debris → netherite scrap. Endgame processing |
| CM1 Demolisher | cut + rocksmash | Universal gatherer — breaks ANY block type. The ultimate general-purpose worker |
| F12 Aura Master | calmmind + helpinghand | Speed + Strength + Haste + Resistance all at once. Walk-by god buff |
| F11 Full Restore | healbell + aromatherapy | Regen II + clears every negative effect. Best healing in the mod |

**8 jobs. All combo. All worth building a Pokémon around.**

## A — Core Automation

Resources you burn through constantly, effects you want active 24/7, or jobs that
eliminate a major pain point. These are the first jobs worth setting up on any server.

| Job | Why A-Tier |
|---|---|
| A12 Apricorn Shaker | **Pokéballs require apricorns.** This is THE Cobblemon resource. Auto-farming apricorns is priority #1 |
| A27 Tumblestone Breaker | **Pokéballs require tumblestones.** Second half of the Pokéball recipe. Must-have |
| A11 Berry Picker | Berries for healing, EV training, held items. Cobblemon's most versatile consumable |
| A13 Mint Harvester | Mints change natures for competitive. Rare, slow-growing, extremely valuable |
| C1 Ore Smelter | Raw iron/gold/copper → ingots with no furnace. The ore→ingot pipeline is constant |
| C2 Food Cooker | Auto-cooks every meat/fish/potato. Never stand at a furnace again |
| A1 Overworld Logger | Wood is the single most used resource in Minecraft. Infinite oak/spruce/birch |
| A8 Grain Reaper | Wheat for bread, hay bales, breeding. Non-stop demand |
| A9 Root Digger | Carrots + potatoes = best renewable food. Always needed |
| A21 Stone Breaker | Stone/cobblestone in bulk. Building, crafting, smelting — always in demand |
| F1 Healer | Passive Regeneration near your base. The original killer feature of the mod |
| D1 Torch Lighter | Auto-lights dark blocks. Spawn-proofs your base without manual torch spam |
| E1 Guard | Kills hostile mobs around your pasture. Real defense, real protection |
| B21 Static Generator | Redstone dust on demand. Every farm, piston door, and circuit needs it |
| CM16 Magma Diver ★ | Obsidian + magma cream + tiny netherite scrap chance. Strong combo |
| CM15 Deep Sea Trawler ★ | Nautilus shells + heart of the sea (very rare). Unobtainable any other way via automation |
| CM18 Gem Vein Finder ★ | Emeralds, diamonds, lapis from excavation. Premium mining combo |

**17 jobs (14 single + 3 combo). The backbone of any Cobbleworkers setup.**

### Implementation Priority

Build S-tier combos last (Phase 7) but design the system so they Just Work when
combo detection is added. Focus implementation effort on A and B tier jobs first —
they're the ones players will set up immediately and judge the mod by.
